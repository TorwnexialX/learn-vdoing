            类的使用

            # 1. 类的基本说明

            应用步骤：

            1. 进行抽象
            2. 声明类
            3. 实现类
            4. 使用类(建立在实例化的基础上) 类与对象：
            5. 类是一种数据类型，以类为数据类型的变量称之为类的实例或对象
            6. 对于同一个类下的不同对象，其数据成员间独立复制，但定义的函数成员共享
            7. 定义对象的一般形式：
            ClassName object_name;
类名 对象名;
            1. 通过对象来访问对象成员：
            object_name.public_data_member;
对象名.公有数据成员;
object_name.public_function_member(real_argument_list);
对象名.公有成员函数名(实参列表)
            1. 通过对象指针访问对象的成员
            指向对象的指针 -> 公有成员函数名(实参列表);
指向对象的指针 -> 公有数据成员;

            [注]：

            1. 四五条访问方式只能访问对象的公有成员
            2. ::用于类，.用于对象，而且实操时一定要想清楚要用的是抽象的类的概念还是要用具体的对象作为操作目标

        cout << Hour + 1 << endl;
    }
    void stem(){
        cout << Hour << endl
    }
};

    Clock aclock;
    aclock.test(5);//输出：6
