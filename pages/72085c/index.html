            this指针

            • C++为每个非静态成员函数提供一个this指针
            • this指针是一个隐含的指针，它指向了正在被成员函数操作的那个对象，且该指针的值不能被改写
            • this指针不能显式声明，它只是非静态成员函数的一个形参
            class Counter{
  int value;
public:
  void setValue( int value ){
	  this->value = value;
  }
  ……
};

void main(){
   Counter obj1,obj2;
   obj1.setValue(1);//该对象中setValue()内的this指针指向obj
   obj2.setValue(2);
}
