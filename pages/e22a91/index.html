            函数重载

            # 1.定义

            区分函数的不同，是区分 函数名、返回值、参数列表(数量&类型) 的不同 而 函数名相同 && 参数列表不同 的情况被称为函数重载

            # 2.区分重载函数

            虽然函数名相同，但OS会根据参数列表的不同自动分配对应函数

            # 3.参数的缺省与默认

            void fun1(int a = 10, double b = 3.14){
...
}
void fun2(char a, int b = 3);

            fun1()的a, b两参数有默认值， fun2()的第二参数有默认值，但注意 默认值只需要在声明处写，定义处再写会报错(指先声明原型后定义的情况)

            int main(){
	fun1();//pm1默认，pm2默认
	fun1(1);//pm1重赋，pm2默认
	fun1(1, 3.0);//pm1重赋，pm2重赋
}

            可见参数的传入是从左向右的 ATTENTIOIN：

            1. 有默认值的形参与重载函数结合使用会导致调用不明确，如
            void fun(int a);
void fun(int a, double b = 2.9);
此处若用fun(1)，会让OS不知道是第一个函数还是第二个函数
            1. 返回值不能作为重载的条件
            2. 形参为float, double（包括其他类似类型）互为重载的函数，调用时要注意形参形式
            void fun(float a);
void fun(double a);

            如果输入fun(1.1F)则调用的是第一个重载函数，输入的是fun(1.1)则用的是第二个重载函数
